About Pakistan

Pakistan is a country located in South Asia. It shares borders with India, Afghanistan, Iran, and China. Known for its diverse landscapes, Pakistan includes vast deserts, mountain ranges, and beautiful beaches.

Capital City

The capital city of Pakistan is Islamabad, known for its modern architecture and high standard of living.

Major Cities

  • Karachi
  • Lahore
  • Islamabad
  • Rawalpindi
  • Faisalabad

Popular Landmarks

Pakistan is home to many historical and natural landmarks, such as:

  • Badshahi Mosque in Lahore
  • Minar-e-Pakistan in Lahore
  • K2 Mountain
  • Moenjo-daro ruins
  • Makli Necropolis






HTML Tags Used

<!DOCTYPE html>

Defines the document type and version of HTML (HTML5).

<html>

Root element that contains all HTML code.

<head>

Contains meta-information about the document (like title, character set, and CSS styles).

<meta>

Provides metadata, which helps with responsiveness and character encoding.

<title>

Sets the page title, displayed on the browser tab.

<style>

Internal CSS styling for layout, colors, fonts, and animations.

<body>

Contains the main content visible on the webpage.

<header>

Top section that displays the title or welcome message.

<section>

A general sectioning tag, used for grouping content into sections for readability.

<h2>, <h3>

Heading tags that organize content hierarchically, giving structure to titles and subtitles.

<p>

Paragraph tag, used for main textual content.

<ul>

Unordered list, ideal for listing items like cities and landmarks.

<li>

List item, used within <ul> to list individual items.

<footer>

Marks the footer section, commonly used for copyright or site information.
